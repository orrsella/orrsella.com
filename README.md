# orrsella.com

Jekyll sources for [orrsella.com](http://orrsella.com).

## Requirements

* [Jekyll 2.2+](http://jekyllrb.com/) - `$ gem install jekyll`
* jekyll-sitemap

## Build

```bash
$ jekyll build
```

## Serve

```bash
$ jekyll serve
```

## Deployment

Usually deployed using [mah-server](https://github.com/orrsella/mah-server).
