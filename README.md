# orrsella.com

Jekyll sources for [orrsella.com](http://orrsella.com).

## Requirements

* [Jekyll 2.2+](http://jekyllrb.com/) - `$ gem install jekyll`
* [Bundler](http://bundler.io/)

```bash
$ gem install bundler
$ cd orrsella.com/
$ bundle install
```

## Build

```bash
$ jekyll build
```

## Serve

```bash
$ jekyll serve
```
